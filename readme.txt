== Description ==
Generates one or more randomly selected paragraphs of Lorem Ipsum text from shortcode. Use shortcode [lorem] for one paragraph. Or add para= X to generate X number of paragraphs. Example [lorem para=3] for 3 paragraphs.

== Installation ==
Upload the full directory, with the folder, into your ‘wp-content/plugins’ directory
Activate the plugin through the ‘Plugins’ menu in WordPress
Use[lorem] shortcode anywhere you want a paragraph of lorem ipsum text.
